# 🧮 Calculadora

Calculadora funcional feita com HTML, CSS e JavaScript puro.

## Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Porcentagem e inversão de sinal
- Display acumulativo de números
- Botão de limpar

## Tecnologias

- HTML5
- CSS3 (Grid Layout)
- JavaScript

## Como usar

Abra o `index.html` no navegador ou acesse pelo GitHub Pages.
